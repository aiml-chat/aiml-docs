# Contributing to aiml-docs

Thanks for helping improve the AIML.chat documentation!

## What lives here

- `docs/getting-started.md` — Registration, indexing, embedding guide
- `docs/api-reference.md` — All API endpoints with curl examples
- `docs/widget.md` — Widget customisation, CSS variables, JS API
- `docs/wordpress.md` — WordPress plugin installation and config
- `docs/faq.md` — Common questions and troubleshooting

## How to contribute

1. Fork the repo and create a branch
2. Edit the relevant `.md` file
3. Open a pull request — describe what you changed and why

## Style guide

- Use `##` for top-level sections, `###` for subsections
- Code examples use fenced code blocks with language hints (` ```bash `, ` ```json `)
- API paths in code: `POST /v1/chat`
- Keep examples minimal — show the happy path, not every edge case
- Link to the live API at `https://api.aiml.chat` not `localhost`

## What we accept

- Corrections to endpoint documentation
- Improved installation instructions
- New FAQ entries
- Translations (create `docs/[lang]/` subdirectory)
